F1: Show number of ratings

As a user, I want to see the number of ratings in order to feel compelled to rate, too.

Examples

Given there are 0 ratings,
When I use the application,
Then I see that there are 0 ratings.

Given there are 3 ratings,
When I use the application,
Then I see that there are 3 ratings.

Given there are 5 ratings,
When I use the application,
Then I see that there are 5 ratings.
